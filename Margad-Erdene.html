Гүйлтэнд гарахдаа хамгийн хэцүү нь гарах шийдвэр гаргах. Гараад 5 мин алхавал бие өөрөө гүйх болдог.
⚡
Хурдаа тогтоох
Ярьж чадах хурдаар гүй. Хэт хурдан гүйхээс болж зогсдог. Удаан, тогтвортой = 25 мин дуусгана.
😴
Нойр = жин хасах
7.5 цаг нойр авбал cortisol буурч жин хасах эрчим 40% нэмэгдэнэ. Гүйлт + нойр = хамгийн хурдан арга.
🥚
Гүйлтийн дараах хоол
Гүйлтийн дараа 30 минутын дотор уураг ид: өндөг, тараг, сүү. Булчин сэргэнэ, ядрал хурдан арилна.
📱
Гүйлтэнд утас байхгүй болго
Зөвхөн дуу сонс. Scroll хийхгүй. Гүйлт бол тархийн "reset" цаг — утас байхгүй үед л ажилладаг.
📈
80кг → зорилго тавих
3 сарын тогтмол гүйлт + хоол хязгаарлах = 5-8кг хасах боломжтой. Тоо биш, тогтмол байдлыг хэмж.
YouTube channel эхлүүлэх практик зөвлөмж
🎬
Эхний 10 видео = сургалт
Эхний 10 видео хэн ч үзэхгүй болно гэж бод. Гол нь техник, дуу, монтажаа сурах — тогтмол нийтлэх.
📝
Script = айхгүй байх
Script бичсэн хүн камерын өмнө айдаггүй. Script = баталгаа. Богино 5 мин видео ч script хэрэгтэй.
🖼️
Thumbnail = 80% клик
Хүмүүс thumbnail харж клик хийдэг. Тод өнгө + нүүр + 3 үгний гарчиг = хамгийн энгийн шийдэл.
📅
7 хоног тутамд нэг видео
Оюутан байж channel хийхэд 7 хоногт нэг видео хангалттай. Тогтмол байдал > их гарах. YouTube алгоритм тогтмолийг шагнадаг.
Маргад-Эрдэнэгийн хуваарийн сайжруулалтууд
01
Нойрны цагийг тогтоох
21:00-22:00 хооронд унтдаг — тогтворгүй
→ 21:30 яг тогтмол унтах. 22:00-д утас хаах.
09:00-19:00 хичээлийн дараа бие ядарна. 21:30 унтвал 07:00 босоход 7.5 цаг нойр хангалттай авна. Нойр = жин хасах + тархи сэргэх + YouTube-д шинэ санаа.
02
Гүйлтийг тогтмолжуулах
7 хоногт 1-2 удаа санамсаргүй дасгал хийх
→ Да/Лх/Бм 07:15 гүйлт. Мя/Пү/Ба/Ня алхалт/амрах.
Оройн дасгал сургуулийн ядралтаас болж тасалддаг. Өглөөний гүйлт = хэзээ ч тасалдахгүй. 3 сарын дараа 80кг → мэдэгдэхүйц өөрчлөлт гарна.
03
YouTube-ийн тодорхой зорилт тавих
"YouTube хийх" гэсэн ерөнхий зорилттой байх
→ "Маргааш X видеоны script дуусгана" гэсэн нэг тодорхой зорилт өдөр бүр бичих.
Ерөнхий зорилт тархийг тогтворгүй болгодог. "Script дуусгах" = тархи шууд ажилладаг. YouTube channel ингэж өсдөг.
04
Хоол урьдчилан бэлдэх
Өдөр бүр хоол бэлдэж цаг алддаг
→ Бямба Да-Пүрэвийн хоолыг, Пүрэв Ба-Нягийн хоолыг batch cook хийх.
09:00-19:00 сургуулийн дараа хоол хийх цаг байхгүй. Batch cook = оройн YouTube цагийг хамгаалдаг. Жин хасахад ч тогтмол эрүүл хоол чухал.
05
Бямбагийн цагийг YouTube-д зориулах
Амралтын өдөр бүрэн тайвших, YouTube хийхгүй байх
→ Бямба 08:30–14:00 нь YouTube-ийн их цаг. Орой нь бүрэн амарна.
Сургуульгүй, тархи шинэхэн = жилийн хамгийн сайн бүтээлч цаг. Оюутан байж channel босгоход амралтын өдрүүд шийдэх үүрэгтэй.
